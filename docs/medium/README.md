## Medium Publishing Steps

### 1. Copy Content to Medium

1. Log in to [Medium.com](https://medium.com/)
2. Click "Write" in the top right
3. Paste the markdown content
4. Medium will auto-convert most formatting

### 2. Upload Images

You need to upload these GIF files from the jenv repository:

**Required GIFs:**
- `jenv.gif` - Insert after "The Solution" section
  - URL: `https://raw.githubusercontent.com/WhyWhatHow/jenv/main/assets/jenv.gif`
  - Caption: "Watch JEnv scan and switch Java versions in real-time"

- `jenv-add.gif` - Insert in Step 4 of the tutorial
  - URL: `https://raw.githubusercontent.com/WhyWhatHow/jenv/main/assets/jenv-add.gif`
  - Caption: "Adding Java versions is this simple"

**How to add images in Medium:**
- Click the "+" button on the left margin
- Select "Image"
- Either upload from computer or paste image URL
- Add caption below image
