## 📖 写作风格指南

### 混合型风格实现

**故事部分**(引入):
- 第一人称叙事
- 具体时间、地点、情绪
- 使用对话和场景描写

**技术部分**(主体):
- 图文并茂(代码 + 截图 + 流程图)
- 类比简化(改路标 vs 搬大楼)
- 数据支撑(300ms, 10x)

**实用部分**(落地):
- Step-by-step 教程
- 场景化应用
- 常见问题 FAQ
- 行动号召 CTA

### 中英文差异

**中文特点**:
- 情感表达更细腻("淋过雨想撑伞")
- 使用成语、网络用语("慢到怀疑人生")
- 掘金文化:技术深入 + 实战导向

**英文特点**:
- 直接、简洁("Dream big, start small")
- 数据驱动("10x faster", "300ms")
- Medium/Dev.to:故事性 + 可读性
